# 🩸 BloodSathi

A full-stack blood donation platform that connects donors and recipients in real-time, improving accessibility and response time during emergencies.

---

## 🚀 Features

- 🔍 Search and request blood donors بسهولة
- 📍 Real-time tracking of blood availability
- 🏥 Hospital inventory management system
- 🔐 Secure authentication using JWT
- ⚡ Fast and responsive UI

---

## 🛠 Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Authentication:**
- JWT, bcrypt.js

---

## 📊 Impact

- 👥 Supports 200+ users for blood requests and donations  
- ⏱ Reduced donor-recipient matching time by ~25%  
- 📈 Improved coordination efficiency by 30%
